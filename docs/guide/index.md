## 安装

浏览器环境:

```js
<script src="flight.js"></script>
```

通过 npm:

```bash
$ npm i -g npm
$ npm i --save flight
```

Nodejs:

```js
// Load the full build.
var flight = require('flight');
// Load method categories.
var array = require('flight/array');
```

ES6:

```js
import { array } from 'flight/array';
```
